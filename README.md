# 🌐 Portfolio Trần Trung Dũng

Đây là website portfolio cá nhân của tôi.  
Trang web được xây dựng bằng **HTML và CSS** để giới thiệu bản thân, năng lực và các dự án của tôi.

---

## 👤 Giới thiệu

Xin chào, tôi là **Trần Trung Dũng**.  
Hiện tôi là học sinh lớp **11B4 – Trường THPT Tôn Đức Thắng**.

Tôi quan tâm đến **lập trình web và công nghệ**, và đang học cách xây dựng website bằng HTML, CSS và các công nghệ liên quan.

---

## 🚀 Công nghệ sử dụng

- HTML
- CSS
- GitHub
- GitHub Pages

---

## 📂 Nội dung website

Website gồm các phần chính:

- **Giới thiệu** – Thông tin cá nhân  
- **Năng lực** – Các kỹ năng và khả năng  
- **Dự án** – Một số sản phẩm đã thực hiện  
- **Liên hệ** – Thông tin liên lạc
